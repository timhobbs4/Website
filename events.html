SAP Inside Track (SIT) Brisbane

June 27, 2019

There has been a while since the last SIT BNE in February, so to make it up to everyone we organised two great speakers for the second SIT BNE for 2019.
Tonight we had Leigh Mason and Jon Gooding, two very experienced people from within the SAP community, who came in to talk about some exciting technologies.
Leigh was first up and he was talking about Fiori Elements. If you have not seen a Fiori Elements development from end to end (OData Annotations on SAP Gateway using CDS Views to SAP Web IDE Fiori Elements Template) this session was a great introduction to get you up and running with the basics.
Jon Gooding then took the stage to talk through chatbots and conversational AI, which was very exciting to hear about. Follow the link to find some more details around conversational AI.

SAP Inside Track (SIT) Brisbane

February 27, 2019

We kicked off the first SIT BNE for 2019 with a great turn-out from the local community.
Tonight we had the pleasure of hosting Matt Coote and Andrew Dyer.
Matt answered the question "Can DevOps work in the world of SAP?......" Surprisingly, there are several places in an SAP project where supplementing your project team with some simple open-source tools can help make development more efficient and project outcomes more reliable.
While Andrew walked us through the work he did to find the cheapest and lowest impact way to get mode 2 style reporting.

Matt Coote showcasing the variety of tools that can be utilised in yourr toolchains.

SAP Inside Track (SIT) Brisbane

November 14, 2018

Another exciting SIT BNE being held at the D'alRae office in Brisbane.
